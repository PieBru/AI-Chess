1. Opening the app

1

Find the file — it's a single page named something like chess.html.

2

Double-click it. It opens directly in your default web browser — Chrome, Firefox, Safari, or Edge all work.

3

That's it. There's nothing to install and no account to create.

Tip: Human vs. Human and Human vs. Normal AI work with no internet connection at all. Only Grandmaster mode (the first time you use it) and LLM-Assisted mode need to reach the internet.

2. Setting up a game

When the app opens, you'll see a setup screen with two panels — one for White, one for Black. For each side, choose one of four options:

OptionWhat it means
HumanYou (or whoever's sitting at the keyboard) make the moves.
Normal AIA built-in computer opponent. Pick a difficulty from 1 (Beginner) to 5 (Expert).
GrandmasterThe strongest setting — runs the Stockfish engine, one of the strongest chess players in existence, human or computer.
LLM-AssistedAn AI language model plays this side. You'll need to supply a few details — see section 6.

You can mix and match freely — a Human vs. a Normal AI, two different AIs against each other, or even the same AI type at two different difficulties, playing itself.

Once both sides are set, press Start Game.

3. Making moves

1

Click the piece you want to move. Its legal destination squares light up — a small dot for a quiet move, a ring around the square for a capture.

2

Click a highlighted square to move there. Click the same piece again to deselect it.

3

If a pawn reaches the far end of the board, a small menu pops up asking which piece to promote to — Queen, Rook, Bishop, or Knight.

4. Reading the board

The eval bar (the vertical gauge beside the board) shows who's ahead. A taller light fill means White is doing better; a taller dark portion means Black is. It updates after every AI move.

Move-quality tags appear next to AI moves in the history panel, so you can see how good each move really was:

Best — the strongest move available.
Good — a small step off the top choice, nothing to worry about.
Inaccuracy — a noticeably better move existed.
Mistake — a costly slip.
Blunder — a serious error, often losing material.

This is a great way to actually see the difference in strength between, say, Normal AI on Beginner and Grandmaster mode — watch the tags, not just who wins.

5. Understanding Normal AI's difficulty levels

LevelFeel
1 — BeginnerMakes mistakes on purpose. Good for a first-ever game.
2 — EasyA relaxed, casual opponent.
3 — MediumA solid club-level challenge.
4 — HardA strong, focused opponent — expect it to punish mistakes.
5 — ExpertThe strongest this hand-built engine can play.

Grandmaster mode has no difficulty setting — it always plays at full strength.

6. Using LLM-Assisted mode

This option lets an AI language model play a side, via any service that speaks the common "OpenAI-compatible" chat format. To set it up you'll need:

FieldWhat to put there
API base URLThe web address of the AI service, e.g. https://api.openai.com/v1, or the address of a program running on your own computer.
API keyA password-like code the service gives you. Leave this blank if you're using a local service that doesn't require one.
ModelThe specific model's name, e.g. gpt-4o-mini.
Privacy note: the app talks to that address directly from your browser — there's no in-between server. Don't paste an API key into a copy of this file that you then share with someone else, since it would travel with it.
If it fails: a bad key, a typo in the address, or a service hiccup won't end your game — the app quietly makes a fallback move and tells you what happened, so play continues.

7. Game controls

ButtonWhat it does
Flip boardTurns the board around, so you can view it from the other side.
ResignEnds the game immediately in your opponent's favor.
New gameReturns to the setup screen. Nothing from the current game is saved.

8. Troubleshooting & FAQ

Grandmaster mode won't load

It needs an internet connection the first time you use it, to fetch the Stockfish engine. If you're offline, or a firewall is blocking it, you'll see a message explaining that, and the option will be disabled for that session.

Can I save a game and come back later?

Not currently — closing or refreshing the page resets everything. Each session starts fresh by design.

Why did the "thinking" indicator take a few seconds?

Higher Normal AI difficulties and Grandmaster mode both spend real time calculating. The board stays fully responsive while this happens — you can still flip it or scroll the move list.

Does this app track or store anything about me?

No accounts, no analytics, nothing saved between sessions. The only outside connections it ever makes are to fetch the Grandmaster engine (if you use that mode) and to whichever address you provide in LLM-Assisted mode.
